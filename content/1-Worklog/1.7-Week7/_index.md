### Week 7 Achievements:

#### Knowledge

**What is AWS Support?**

- Understand that AWS Support is a service that helps customers use AWS services effectively.
- Learn that AWS Support provides different support levels tailored to various customer needs.
- Understand that AWS Support assists with technical, billing, and operational issues.

**AWS Support Plans**

- Understand the available support plans:
  - Basic Support
  - Developer Support
  - Business Support
  - Enterprise Support
- Learn the scope of support and response times for each plan.
- Understand that Basic Support is provided free of charge to all AWS customers.

**AWS Support Center**

- Learn how to access AWS Support Center from the AWS Management Console.
- Understand how to manage and track Support Cases.
- Learn how to review support history and case status.

**Technical Support Cases**

- Learn how to create technical support requests.
- Understand how to provide clear issue descriptions for effective assistance.
- Learn how to select the appropriate severity level.

**Billing and Account Support**

- Understand how to submit support requests related to billing and account management.
- Learn how to handle AWS billing issues.
- Understand the communication process with AWS Billing Support.

**AWS Trusted Advisor**

- Understand that AWS Trusted Advisor analyzes AWS environments and provides best-practice recommendations.
- Learn that Trusted Advisor covers:
  - Cost Optimization
  - Security
  - Performance
  - Fault Tolerance
  - Service Limits
- Learn how to review and apply recommendations.

**Case Management**

- Learn how to monitor the progress of Support Cases.
- Understand the communication workflow between customers and AWS Support.
- Learn how to close Support Cases after issues are resolved.

- Access AWS Support Center.
- Review existing Support Cases.
- Explore the Create Case feature.
- 📸 _Evidence: AWS Support Center interface._

##### Module 4 — Create a Technical Support Case

- Select **Create Case**.
- Choose **Technical Support**.
- Select the affected AWS service.
- Enter the Subject and Description.
- Submit the support request.
- Track the case status.
- 📸 _Evidence: Technical Support Case creation._

##### Module 5 — Billing Support

- Select **Create Case**.
- Choose **Account and Billing Support**.
- Enter the billing-related issue details.
- Submit the Billing Support Case.
- Review AWS responses.
- 📸 _Evidence: Billing Support Case._

##### Module 6 — AWS Trusted Advisor

- Access AWS Trusted Advisor.
- Review optimization recommendations.
- Record findings related to cost and security.

### Challenges Encountered:

- Difficulty distinguishing the scope of each AWS Support Plan.
- Some features are only available with paid support plans.
- Issue descriptions must be detailed to receive accurate assistance.
- Trusted Advisor provides numerous recommendations that require additional study.

---

### Solutions:

- Carefully review AWS Support documentation before performing labs.
- Compare support plans to better understand their benefits.
- Prepare complete issue details before creating Support Cases.
- Regularly review Trusted Advisor recommendations to identify potential issues.
- Save AWS Support responses for future reference.
